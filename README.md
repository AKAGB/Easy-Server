# 利用Node.js实现简单的服务器程序

通过一个简单的例子学会使用Node.js常用的一些模块和函数，创建的服务器程序可以提交表单，并将数据进行直接展示。

教程是[Node 入门](https://www.nodebeginner.org/index-zh-cn.html)学习，适合具有一定javascript基础以及对http协议有所了解的人。
